# Scoop Bucket for agentio

CLI for LLM agents to interact with communication and tracking services.

## Installation

```powershell
scoop bucket add agentio https://github.com/plosson/scoop-agentio
scoop install agentio
```

## Upgrade

```powershell
scoop update agentio
```

## Usage

```powershell
agentio --help
```
